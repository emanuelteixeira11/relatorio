Este conjunto de ficheiros pretende facilitar a escrita, de acordo com o formato requerido para os trabalhos académicos no Instituto Politécnico de Beja e, em particular, para os relatórios de estágio ou projecto final de curso da licenciatura em Engenharia Informática da Escola Superior de Tecnologia e Gestão do mesmo instituto. Assim, destinam-se à unidade curricular de "Estágio ou Projecto" do último semestre desse mesmo curso.

Possivelmente com muito pouco esforço adicional, serão adaptáveis a outros relatórios de projectos finais ou documentos semelhantes.

O formato segue (ou pelo menos está muto próximo) do recomendado pelo respectivo documento normativo do IPBeja intitulado " Normas Obrigatórias para a Elaboração de Documentos Institucionais e Trabalhos Académicos".

Os ficheiros tentam ser auto-explicativos, MAS continua a ser necessário possuir ou adquirir conhecimentos básicos de LaTeX...

Aqui ficam as instruções resumidas:

  1. A primeira tarefa será instalar uma distribuição LaTeX no seu computador. As recomendações usuais são as seguintes:
    • Para Windows: http://miktex.org/
    • Para Mac OS: http://tug.org/mactex/ (inclui editores de texto)
    • Para Linux: http://www.tug.org/texlive/ (veja por exemplo o seguinte link: http://xmodulo.com/2013/01/how-to-install-latex-on-ubuntu-or-debian.html)
    Em Windows, os seguintes dois editores são muito utilizados:
    • http://www.texniccenter.org/
    • http://www.xm1math.net/texmaker/
  2. O ficheiro principal tem o nome "minutaDoRelatorio.tex". É este que permite gerar o relatório.
  3. A informação "base" (titulo, autor, etc.) deve ser colocada no ficheiro "info-para-preencher.tex" nos locais lá assinalados.
  4. A bibliografia deve ser colocada no ficheiro "bibliografiaDoRelatorio.bib".
  5. O conteúdo do relatório deve ser colocado em ficheiros nas seguintes directorias:
    • "parte-inicial" resumo, agradecimentos, etc.
    • "capitulos" texto dos capítulos
    • "figuras" para todas as imagens
    • "listagens" para listagens de código
    • "apendices" se existentes
    • "anexos" se existentes
  6. A sequência típica de execução será executar dois programas (latex e bibtex) pela seguinte ordem:
    • latex "minutaDoRelatorio.tex" (executar duas ou três vezes para gerar um "minutaDoRelatorio.pdf" a partir dos ficheiros.tex, imagens e listagens)
    • bibtex (para gerar a bibliografia em "minutaDoRelatorio.bbl" a partir do ficheiro "minutaDoRelatorio.aux" e do ficheiro "bibliografiaDoRelatorio.bib")
    • latex (executar duas ou três vezes para gerar o "minutaDoRelatorio.pdf" já com a bibliografia completa). Se já tiver a bibliografia gerada (o ficheiro "minutaDoRelatorio.bbl"), bastará o passo 6.1.
